# CodeKata
People Matter Code Kata Submission Repo

## Code submission guidelines

* Solve each kata / assignement in a programming language other than the previous one
* For each submission, insert a file header with your name, date and kata/assignment name
* Your code should be production ready: check input, use appropriate style and code practices, comment if necessary
* You may submit up to 2 different solutions:
 * First solution should use only core language features and consturcts, attempt to only use official documentation for your language of choice, i.e. no stackoverflow
 * Second solution may be submitted to demonstrate little know programming language features, feel free to use any resources avaialabe

## Directory structure

The following directory structure will be used for all submissions

__/Kata Name/Language Name/Your Name/Your file.extension__
